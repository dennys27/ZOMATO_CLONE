
# Z-FASHIONS

A clone website of zomato built with help of react.js and firebase
# Features

User  
1.carousel   
2.product listing  
3.similer to zomato 


# Technologies used   
React.js  
Firebase  
git  




#### Get HomePage

```http
  GET /locahost:3000
```





## Run Locally

Clone the project

```bash
https://github.com/dennys27/ZOMATO_CLONE
```

Go to the project directory
# Client
```bash
  cd zomato-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
